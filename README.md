# Family-Game-Teller-System
This is a bank system for family board games that use a monetary system.
# Functions
/charge                                                                                                                                                                                                              
Charges a player money

/give

Give a Player Money

/transfer

Transfer money from one player to another

/bankcrupt

Set a player's money to $0 Used for trolling or punishment

/status

See stats of a Player

/payjail

Pay to leave jail

/help

Pull up list of commands

/end

End the game
